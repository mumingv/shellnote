# grep

## 基本语法

## 命令参数

###### 
|常量名称			    |含义						|
|-----------------------|---------------------------|
|-o, --only-matching    |只输出匹配到的字符串		|
|-E, --extended-regexp  |扩展的正则表达式           |

	
## 命令示例

###  

#### 示例：提取每行中的logid

```bash
$ cat rac.log | grep -o -E 'logid:[0-9]{14}' | cut -d ':' -f2
```
```bash
15016440214828
15016440227953
15016440222413
15016440221380
15016440234939
15016440245951
```
